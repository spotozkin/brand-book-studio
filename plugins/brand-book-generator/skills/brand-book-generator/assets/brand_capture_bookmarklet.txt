Notes for the skill:
- Reads the page only; displays a textarea; sends nothing anywhere. Say this to the user.
- Cross-origin stylesheets are skipped silently (cssRules throws) — the computed{} block
  still captures real used colors/fonts regardless, so the capture degrades gracefully.
- If the paste-back JSON has an empty tokens{} but a full computed{}, treat computed
  values as class 2 evidence and note that the site defines no public CSS tokens.
